# Lumen RESTful API

### Requirements

* docker

### Project setup

* Add `framey.local` to your `/etc/hosts`: `127.0.0.1 framey.local`

* Run `make init` to initialize project

* Open in browser: http://framey.local:8000/api/documentation

### Run tests

make tests

### Project endpoints

Please see http://framey.local:8000/api/documentation
